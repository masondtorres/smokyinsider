# Homepage Visual Asset Manifest

Verified: 2026-07-21

Canonical source records are stored in `data/homepage-assets.json`. The acquisition workflow downloads only records marked `verified`, `public-domain` or `licensed`. It rejects empty files, non-image responses, MIME mismatches, dimension changes and files larger than 30 MB. After acquisition, this document is regenerated with final file size and SHA-256 values.

## Rights rule

National Park Service photographs in this manifest are credited to NPS and display no copyright symbol on the exact NPS media detail page. The detail pages state that such NPS-credited multimedia is public domain. Media credited to a third party is not presumed public domain.

Creative Commons photographs retain the creator, source page and license link required for attribution.

## Exact records

Each record in `data/homepage-assets.json` includes:

- Local output path
- Homepage role
- Descriptive alt text
- Exact source page
- Direct download URL
- Creator and visible credit
- License and license URL
- Rights basis
- Expected MIME type
- Expected pixel dimensions
- Approval status
